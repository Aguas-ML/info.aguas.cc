Stackspin é uma plataforma que oferece provisionamento autogerenciado, clique e reproduza de aplicativos online para Organizações da Sociedade Civil (OSC). Os usuários poderão configurar facilmente uma instância auto-hospedada do Stackspin, para que possam manter o controle sobre os dados inseridos nesses aplicativos.

Stackspin é:

  • Código aberto
  • Atualização automática
  • Fácil de implantar
  • Integrado

 

Como instalar

https://docs.stackspin.net/en/v0.8/installation/install_cli.html

 

Anotações de uso

 

Novo usuário

Criar nova conta após start

 

apt update
apt upgrade -y
adduser nick
usermod -aG sudo nick
su - nick
sudo nano /etc/hostname
sudo nano /etc/hosts
exit
reboot

 

Faça a instalação do Docker

Entre com seu novo usuário e faça a instalação 

Link tutorial

Depois:

 

sudo usermod -aG docker nick
reboot

 

Ao usar o PIP

PIP, sempre ativar antes de instalar e rodar

 

cd /stackspin
. env/bin/activate
